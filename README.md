# Weather-App

Creating a simple Weather App using HTML, CSS, and JavaScript involves these key steps:

•	**HTML** – for the structure of the app.                                           
•	**CSS** – for styling.                                            
•	**JavaScript** – to fetch weather data from an API and display it.
